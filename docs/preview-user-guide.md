# 搜索结果预览功能用户指南

## 概述

预览功能允许您在搜索结果中快速查看 NFO 文件内容，无需加载到编辑器。

## 使用方法

### 悬停预览（桌面端）

1. 将鼠标悬停在搜索结果项上
2. 等待 300 毫秒
3. 快速预览卡片将显示核心信息：
   - 标题
   - 年份
   - 评分
   - 类型标签

### 侧边栏预览（桌面端）

1. 点击搜索结果中的 "👁 预览" 按钮
2. 右侧将展开预览面板，显示完整信息：
   - 基本信息（标题、原标题、类型、年份、评分、时长）
   - 类型标签
   - 海报路径
3. 可以直接在预览中执行操作：
   - **📥 加载到编辑器** - 将文件加载到主编辑器
   - **📁 打开所在目录** - 导航到文件所在目录

### 移动端预览

1. 点击 "👁 预览" 按钮
2. 全屏预览模态框将打开
3. 下拉手势关闭预览（下拉超过 100px）

## 功能特性

### 缓存机制

- 预览数据自动缓存 5 分钟
- 重复查看同一文件直接从缓存加载
- 最多缓存 50 个文件
- LRU 策略自动清理旧缓存

### 批量预览

- 后端支持一次请求多个文件的预览数据
- 减少网络请求，提高性能

### 响应式设计

- **桌面端**（>1024px）：侧边栏面板
- **平板端**（769px-1024px）：底部抽屉
- **移动端**（≤768px）：全屏模态框

## 键盘快捷键

- `Escape` - 关闭预览面板

## 注意事项

- 悬停预览在移动端不可用
- 预览数据包含核心字段，完整数据请加载到编辑器查看
- 预览不会影响当前编辑器状态
